## 📂 仓库目录结构说明 (Repository Structure)

本仓库采用按技术栈和学习阶段进行分类的目录结构，以便清晰地记录我的前端学习路径。

*   ### `00-portfolio-projects/`
    *   **作品集项目索引**。此目录存放一些早期完成的、可独立展示的综合性项目。为了更好的管理，更大型的项目将会拥有其专属的 GitHub 仓库。

*   ### `01-html-css-foundations/`
    *   **HTML & CSS 基础练习**。存放所有关于 HTML5 和 CSS3 核心知识点的静态页面练习，如页面布局、响应式设计、CSS 动画等。

*   ### `02-javascript-basics/`
    *   **JavaScript 核心与 DOM 编程**。包含原生 JavaScript 基础语法、DOM/BOM API 操作、事件处理等相关的练习项目。
    *   **`00-js-implementations/`**: 此子目录下存放的是对 JS 底层原理的深度实践，如**手写 `Promise`**、防抖、节流等，是体现 JS 内功的部分。

*   ### `03-react-essentials/`
    *   **React 核心概念与实践**。存放所有与 React 框架相关的学习项目和练习。从基础的 Hooks 练习，到使用 Redux 的综合性项目，都归纳于此。

*   ### `learning_logs/`
    *   **我的学习简报**。存放我每日学习后生成的 `.md` 格式学习简报，用于复盘、总结和记录成长。
